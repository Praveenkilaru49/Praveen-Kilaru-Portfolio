Overview

This project explores the stability and control of an inverted pendulum mounted on a cart using a PID controller implemented in MATLAB. It applies control system theory to real-time system stabilization challenges.

Project Description

The inverted pendulum system is a classical benchmark problem. This project models its nonlinear dynamics using Newtonian mechanics. The objective is to stabilize the pendulum in the upright position using a PID controller that applies corrective force to the cart.

MATLAB Simulink was used to simulate the system, showing that proper tuning of PID parameters enables fast and stable correction after disturbances. This project demonstrates key control engineering principles like feedback, error correction, and dynamic response analysis.

Tools Used

  • MATLAB & Simulink
  • Control Systems Toolbox
  • Differential Equation Modeling

Skills Achieved

  • PID Controller Design
  • MATLAB Simulation
  • Nonlinear Dynamic Modeling

Conclusion

The PID-controlled inverted pendulum model effectively stabilizes the system. Simulation results demonstrate that well-tuned PID parameters provide reliable performance and robustness against external disturbances.
