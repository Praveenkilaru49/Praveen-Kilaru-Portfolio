Overview

This project focuses on optimizing 3D printing resolution by applying Anti-Aliasing (AA) and Gaussian blurring to binary mask images. By improving edge transitions in mask patterns, the fidelity and smoothness of 3D-printed surfaces can be significantly enhanced.

Project Description

The study aims to improve the accuracy and resolution of 3D-printed components by enhancing the quality of binary mask images used in the printing process. This project addresses aliasing effects through Gaussian blurring and edge smoothing.

Various kernel sizes were tested and analyzed using Python and MATLAB. Experimental validation included scanning electron microscopy. An outlook on ML-driven automation for AA parameter selection was also explored.
