☀️ Beacon Solar Project + BESS

Overview

This project analyzes the LADWP’s Beacon Solar + BESS deployment in the Mojave Desert, highlighting lithium-ion battery integration, system capacity, operational metrics, and sustainability insights for future grid-scale renewable storage.

Project Description

This project evaluates LADWP's Beacon Solar and Battery Energy Storage System (BESS), featuring a 250 MW solar array and 20 MW / 80 MWh lithium-ion storage. Analysis includes battery configuration, inverter behavior, and charge cycles.

It compares performance with Tesla's Hornsdale Reserve and proposes AI grid control, hydrogen storage, and modular scalability for future resilience.

Tools Used

  • Technical Data Review (LADWP & DOE)
  • BESS Configuration Analysis
  • Performance Benchmarking
  • Policy & Sustainability Frameworks

Skills Achieved

  • Battery Storage Analysis
  • Grid-Scale Integration
  • Lifecycle and Recycling Strategy
  • Sustainable Energy Planning
